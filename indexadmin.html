☁️ Cloud Sync Setup

Connect to JSONBin.io — free cloud JSON storage. No database needed. One-time setup.

Step 1: Go to jsonbin.io → Sign up free (Google login works)
Step 2: Click "API Keys" in top menu → copy the Master Key (long string starting with $2b$)
Step 3: Click "Bins" → "Create a Bin" → paste {"matches":[]} → click Create
Step 4: In the new bin, click "Share" or the settings icon → set it to Public
Step 5: The Bin ID is in the URL: jsonbin.io/b/THIS_PART
Step 6: Paste both below and click Test & Save
